Comparison: Custom CSS(Responsive) vs Bootstrap - Hungary

Implementation Comparison

Implementing the "Discover Hungary" webpage using custom CSS (Part A) and Bootstrap (Part B) presented a few challenges and benefits. With custom CSS, I had to spend more time setting up the layout and styling everything from scratch. It was a bit tricky but gave me a lot of control over how things looked. On the other hand, using Bootstrap was much quicker. It has ready-made components that made building a responsive design easy

I found the Bootstrap version to be easier and faster to work with. The documentation for Bootstrap was really helpful, which made fixing issues simpler. With Bootstrap (Part B), things were much quicker. I used Bootstrap's grid system with classes like row and col to create responsive layouts easily. The card component was super helpful for displaying attractions - I just had to use card, card-body, and card-title classes to get a nice, consistent look. I also used Bootstrap's utility classes like text-center and mb-4 for quick styling and spacing. While it was faster and easier with Bootstrap, I felt like I had less control over the fine details of the design compared to the custom CSS version.
